About Smart Transit System

Smart Transit System is an AI-powered bus ETA prediction system.

Problem

PMPML commuters have no easy way to know when the next bus will arrive.

Solution

Select boarding and destination stops. Only directly connected buses shown.

Technologies Used

Python 3Flaskflask-cors scikit-learnNumPyPandas Leaflet.jsOpenStreetMap GPS APIHTML5 / CSS3 / JS

How the ETA Model Works

Linear Regression on distance, stops, and traffic. Peak hours 1.40× delay.

Future Enhancements

Live PMPML GPS tracking, real GTFS feeds, weather delays, push notifications.
